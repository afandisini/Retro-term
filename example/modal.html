MODAL_VARIANTS

Demo modal ini menampilkan tiga ukuran: medium, extra large, dan fullscreen. Semua body modal dapat di-scroll tanpa mengganggu layout utama.

INSTALLATION

Pola dasar untuk memanggil modal:

<button class="term-btn term-btn-view" type="button" data-term-action="open-modal" data-term-target="modalDemoMd">
  OPEN_MD
</button>

Pakai class tambahan di dialog untuk ukuran yang berbeda:

<div class="term-modal-dialog term-modal-dialog-xl">
  <div class="term-modal-body">...</div>
</div>
