CARD DEMO

Sidebar demo tersedia di kiri untuk berpindah ke halaman komponen lain.

Halaman ini disusun sebagai referensi kartu yang siap dipakai ulang di halaman lain.

VARIANT GRID
PRIMARY
Blueprint
Cocok untuk highlight utama.
INFO
Update
Dipakai untuk status informatif.
SUCCESS
Stable
Baik untuk indikator sukses.
WARNING
Review
Untuk perhatian atau pending.
DANGER
Alert
Untuk kondisi penting atau error.
NEUTRAL
Base
Untuk konten yang lebih tenang.
MEDIA_CARD
TOP
Responsive Hero
Gambar penuh di atas, konten card di bawah.
LEFT
Side Layout
Cocok untuk ringkasan, listing, atau media object.
RIGHT
Side Layout
Pada mobile akan otomatis stack ke bawah.
COVER
Full Bleed
Pakai untuk hero card atau banner visual.
INSTALLATION

Pola pemasangan card yang disarankan:

<div class="term-card term-bg-success">
  <div class="term-card-header">
    <span class="term-card-title">CARD TITLE</span>
  </div>
  <div class="term-card-body">
    <p class="term-mb-2">Konten card</p>
    <button class="term-btn term-btn-success">Action</button>
  </div>
</div>
