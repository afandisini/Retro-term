Retro-term is a retro-modern CSS framework designed for building beautiful dashboards, admin panels, and web applications with a unique terminal-inspired aesthetic combined with modern design principles.
Retro-term includes buttons, forms, rt-tables, rt-modals, toasts, rt-alerts, rt-dropdowns, accordions, carousels, rt-navbars, rt-cards, and many more UI components to build complete interfaces.
Yes! All Retro-term components are fully responsive and work seamlessly across desktop, rt-tablet, and mobile devices. The framework includes mobile-specific optimizations and touch-friendly interactions.
