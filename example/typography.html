p, b, strong, em, small, mark, u, code

Ini adalah contoh paragraf biasa dengan teks bold, strong, italic, underline, dan highlight.

Retro-term cocok untuk dashboard yang butuh teks tegas dan rapi. Ini contoh teks kecil untuk catatan tambahan.

Inline code bisa dipakai seperti .rt-badge--success atau .rt-tbl-primary.

Typography yang baik membuat isi lebih mudah dipindai, lebih nyaman dibaca, dan lebih konsisten di seluruh dashboard.

Gunakan bold untuk penekanan, emphasis untuk ritme, dan underline untuk penanda penting.
